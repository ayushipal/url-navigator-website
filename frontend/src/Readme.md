# Url Navigator

A full-stack web application that allows users to upload CSV/Excel files or Google Sheets containing website URLs and navigate between them.


## Features

- Upload CSV / Excel files
- Load URLs from Google Sheets
- Display websites inside the app
- Navigate using Next / Previous buttons
- Close iframe viewer
- Responsive UI

---

## Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Libraries: Axios, Multer, XLSX
